
A clean, professional, and responsive portfolio website showcasing my skills, experience, projects, and contact information.

## Project Information

This is a personal portfolio website built with HTML and CSS, featuring a modern design with dark mode support, smooth animations, and full responsiveness across all devices.

## Technologies Used

- **HTML5** - Structure and semantic markup
- **CSS3** - Styling with Flexbox for layout
- **Font Awesome** - Icons for social media and UI elements
- **JavaScript** - Dark mode toggle functionality

## Features

- ✅ Responsive design (mobile and desktop)
- ✅ Clean and professional layout
- ✅ Flexbox for flexible layouts
- ✅ Consistent colors and fonts throughout
- ✅ Dark mode toggle
- ✅ Smooth scroll animations
- ✅ CV download option
- ✅ Contact form
- ✅ Social media links

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```

3. Open `index.html` in your web browser

4. Make sure to add your CV file (`cv.pdf`) in the `images/` directory for the download button to work

## Customization

- Replace `images/l230624_img.jpeg` with your own profile picture
- Add your CV file as `images/cv.pdf`
- Update social media links in the HTML
- Modify colors in `css/style.css` to match your preferences
- Update personal information in `index.html`
